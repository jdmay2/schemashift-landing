SchemaShift

TypeScript schema migration CLI. Convert between Zod, Yup, Joi, io-ts, Valibot, ArkType, Superstruct, and Effect Schema with AST-based transformations.

npm install -g schemashift-cli

Why Migrate With SchemaShift?

Automatic Type Helpers

Converts InferType, TypeOf, z.infer, and v.InferOutput across all supported libraries automatically.

Inline TODO Comments

Unsupported patterns get /* TODO(schemashift): ... */ comments with actionable guidance and links, so nothing is silently dropped.

Form Library Awareness

Detects react-hook-form, Formik, and Mantine imports and warns you to update resolver/adapter packages for the target library.

Zod v3→v4 Coverage

Handles 20+ breaking changes: .merge(), .superRefine(), z.nativeEnum(), .flatten()/.format(), ._def, error params, .pipe(), and more.

Backup & Rollback

Every migration creates a timestamped backup. Use schemashift rollback to restore originals if anything goes wrong.

AST-Based Precision

Uses ts-morph for real TypeScript AST manipulation, not fragile regex. Handles nested schemas, chained methods, and string-literal safety.

Form Resolver Migration

Automatically rewrites @hookform/resolvers imports during migration — e.g., yupResolver → zodResolver. No manual import changes needed.

Complexity Estimation

Pre-migration effort scoring (trivial/low/moderate/high/extreme) with advanced pattern detection, risk areas, and performance warnings.

TODO Resolution Summary

Categorized guidance for all generated TODO(schemashift) comments with actionable resolution steps grouped by type.

Monorepo-Aware Migration

Automatically detects npm/Yarn/pnpm workspaces and suggests topological migration order so dependent packages migrate correctly.

Behavioral Warnings

Detects coercion, validation order, and error format differences between libraries before you migrate. Know exactly what changes.

Bundle Size Estimation

Estimates minified+gzipped size impact of switching schema libraries. See the delta before committing to a migration.

Performance Analysis

Identifies validation speed changes, parse overhead differences, and runtime behavior impacts for your specific migration path.

Type Deduplication

Finds structurally similar schemas across your codebase with confidence scoring. Clean up before or after migration.

Test Scaffolding

Generates validation test stubs for every migrated schema. Verify correctness automatically after migration.

Audit Logging

Records detailed per-file migration history with timestamps, status, and warnings. Full compliance trail for enterprise teams.

Backward Migrations

Migrate back from Zod to Yup or from Valibot to Zod. Full bidirectional support for when requirements change.

Dependency Graph Visualization

Export schema dependency graphs in DOT (Graphviz) or Mermaid format. Color-code by library, highlight circular dependencies, and filter by schema type.

Migration Presets

Built-in templates for common scenarios: React Hook Form + Yup→Zod, tRPC Zod v3→v4, Express Joi→Zod, Next.js forms, and monorepo staged migrations.

Enhanced .when() Migration

Smart Yup .when() analysis generates the right Zod replacement: z.discriminatedUnion(), z.union(), or .superRefine() based on the condition pattern.

Standard Schema Advisor

Detects when both source and target support Standard Schema 1.0. Recommends adapter approach vs full migration with pros, cons, and generated wrapper code.

Approval Workflows

Review gates for shared codebases. Create migration requests, assign reviewers, approve or reject before executing. File-based persistence for auditability.

Webhook Notifications

Send migration events (started, completed, failed, governance violations, drift detected) to Slack, Teams, Zapier, or any HTTP endpoint with HMAC-SHA256 signing.

Cross-Field Validation Helpers

Reusable .superRefine() patterns for common cross-field logic: requireIf, requireOneOf, mutuallyExclusive, and dependentFields.

Project Health Doctor

Run schemashift doctor to diagnose your project: detect schema libraries, flag outdated versions, check form compatibility, and get actionable migration suggestions.

Schema Verification

Contract testing for migrated schemas. Extracts schema names, generates test samples, and verifies migrated schemas produce the same validation behavior as originals.

Canary Migration

Migrate a percentage of files first with --canary 20. Starts with the simplest files, letting you verify before committing to a full migration.

Unified Diff Export

Export standard unified diffs with --output-diff for integration with git apply, code review tools, and CI pipelines. Works with dry-run mode.

Compliance Reports

Generate SOC2 and HIPAA compliance reports from migration audit logs. Includes change control summaries, integrity verification, and access control records.

Multi-Platform CI/CD

Ready-to-use templates for GitHub Actions, GitLab CI, and Azure DevOps. Includes JUnit test reporting, artifact publishing, and governance checks.

Pre-Commit Hooks

Integrate governance checks into your git workflow with schemashift hooks install. Detects husky and lint-staged automatically.

Policy-as-Code

Manage schema governance policies with schemashift policy. Initialize from templates, check compliance, and list active policies for your organization.

Duration Estimation

Get pre-migration time estimates based on complexity analysis. Ranges from minutes (trivial) to hours (extreme) based on schema count and advanced patterns.

Advanced Features

Beyond basic migration. SchemaShift provides deep analysis and enterprise-grade controls.

Advanced Analysis

Deep complexity scoring, migration readiness assessment, and detailed schema introspection to understand migration impact before you start.

Risk Scoring

Per-file risk assessment with configurable thresholds. Export to CSV for project tracking. Fail CI builds that exceed your risk tolerance.

Cross-File Resolution

Dependency graph analysis resolves schema imports across files, ensuring multi-file projects migrate in the correct order.

Chain Migrations

Multi-step migrations like yup→zod→valibot in a single command with automatic intermediate validation.

Compatibility Checking

Pre-migration validation detects version-specific incompatible patterns and provides actionable suggestions before transforming code.

Governance & Plugins

Enforce naming conventions, complexity limits, and validation rules. Extend SchemaShift with custom migration plugins for your organization.

Form Resolver & Complexity

Automatic @hookform/resolvers import rewriting and pre-migration effort scoring with risk area identification and performance warnings.

Monorepo Support

Workspace detection with topological ordering. Migrates packages in dependency order to prevent breakage across your monorepo.

Schema Drift Detection

Snapshot-based comparison detects when schemas diverge from baseline. Track added, removed, and modified schemas over time for compliance and audit.

CI/CD Integration

SARIF output for GitHub Code Scanning, JUnit for test reporting. Templates for GitHub Actions, GitLab CI, and Azure DevOps. Enhanced audit logging with JSON/CSV export.

Governance Templates

Built-in rule templates: no-any-schemas, require-descriptions, max-nesting-depth, naming-convention, and more. Enforce schema quality standards out of the box.

Contract Testing

Verify migrated schemas produce the same validation results as originals. Generates test samples, runs against both libraries, and reports parity scores.

Compliance & Audit

SOC2 and HIPAA compliance reports. Policy-as-code governance. Approval workflows with reviewer tracking. Full audit trail for regulated environments.
